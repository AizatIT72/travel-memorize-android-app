# Android версия приложения Travel Memorize

## **Идея проекта**

**Travel Memorize** - приложение, которое представляет собой интерактивную карту для сохранения истории путешествий и обмена туристическим опытом. Основная концепция заключается в визуализации посещенных мест через метки на карте, к которым пользователь прикрепляет информацию: фотографии, текстовые описания и даты посещения. Приложение позволяет взаимодействовать с другими пользователями, подписываясь друг на друга. Это дает возможность просматривать карты друзей, изучать их рекомендации и находить локации для новых маршрутов. Также пользователь может посмотреть список мест, которые он посещал.

## **Стек технологий**

- Kotlin
- Android API 29
- Clean Architecture, MVVM
- Работа с Room
- Firebase
- Работа с сетью - Retrofit
- Kotlin Coroutines, Flow
- Dagger2
- Навигация: Jetpack Navigation Component
- UI: Jetpack Compose
- Проверка codestyle с помощью Detekt

## **Базовый функционал**

**Авторизация и профиль:**
- Регистрация и вход через email/пароль с обязательной валидацией полей.
- Личный профиль со статистикой (количество стран, городов, воспоминаний) и настройками.

**Интерактивная карта:**
- Отображение текущего местоположения пользователя.
- Добавление меток в местах посещения через кнопку «Добавить».
- Визуальное различие между своими метками и метками друзей.

**Управление воспоминаниями:**
- Создание записи: прикрепление нескольких фото, заголовка, даты и описания.
- Просмотр деталей: экран с галереей фото и текстом воспоминания (появляется при нажатии на метку).
- Редактирование и удаление записей.

**Взаимодействие между пользователями:**
- Поиск друзей по имени/username.
- Возможность подписаться на другого пользователя(добавить в друзья)
- Отписаться от пользователя (кнопка Delete)
- Список друзей и возможность перехода на их интерактивные карты.
